# Support

## Where to Ask for Help

Use these channels to get help quickly and keep issue tracking focused.

- Questions, usage help, and workflow discussion:
  - [GitHub Discussions](https://github.com/NoorXLabs/SkyBox/discussions)
- Bug reports:
  - [Open a bug report issue](https://github.com/NoorXLabs/SkyBox/issues/new/choose)
- Feature ideas:
  - [Open a feature request issue](https://github.com/NoorXLabs/SkyBox/issues/new/choose)
- Documentation gaps or corrections:
  - [Open a docs improvement issue](https://github.com/NoorXLabs/SkyBox/issues/new/choose)

## Issue vs Discussion

Open an issue when you have:

- A reproducible bug
- A concrete feature request
- A specific documentation correction

Use Discussions when you have:

- Questions about setup or usage
- Architecture or design brainstorming
- General community conversation

## Security Reports

For vulnerabilities, do not open public issues or discussions.

Use private reporting as documented in [SECURITY.md](SECURITY.md).
